# Inventory & Billing System

A combined inventory management and billing application using Express and SQLite.

## Setup

1. Install Node.js from https://nodejs.org/
2. Open a terminal in `Inventory and Billing`
3. Run:
   ```bash
   npm install
   npm start
   ```
4. Open `http://localhost:3000` in your browser.

## Features

- Product CRUD management
- Inventory listing and stock updates with search
- Cart-based billing and sale checkout
- Sales history
- Export inventory and sales to CSV or XLS for printing/reporting

## Files

- `app.js` - Express backend and SQLite database logic
- `App.html` - Frontend HTML user interface
- `script.js` - Frontend interaction and API calls
- `package.json` - Dependency manifest
